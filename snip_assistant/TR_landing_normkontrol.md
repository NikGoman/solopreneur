# Техническое задание: Лендинг для валидации MVP «НормКонтроль» (обновленная версия)

## 1. Цель
Создать минимально жизнеспособный лендинг для сбора 50 контактов пилотной группы (прорабы/техдиректоры строительных компаний РФ/СНГ) за 14 дней при бюджете ≤ $15. Фокус: проверка спроса через конверсию в лиды, а не продажи. Лендинг должен:
- Грузиться за ≤1.5 сек (Mobile, 3G)
- Ранжироваться по ключевым запросам о строительных нормах и штрафах
- Автоматически цитироваться в ответах ИИ-чатов (Квен, ChatGPT, Дипсик) при запросах о СНиП
- Содержать виральные элементы для органического распространения в профессиональных сообществах

## 2. Целевая аудитория
Прорабы и технические директора строительных компаний (10–100 сотрудников)
Работают с госзаказами (83% рынка)
Ищут решения для:
- «Как быстро проверить СНиП на объекте?»
- «Чем грозит несоответствие ГОСТ при приемке работ?»
- «Где найти актуальные строительные нормы 2025?»

## 3. Ключевые разделы лендинга (минималистичная структура)

### 3.1. Hero-секция (первый экран)
**Заголовок:** «Ваш личный эксперт по СНиП в Telegram: задал вопрос — получил ссылку на норму за 30 секунд»

**Подзаголовок:** «Забудьте о штрафах за несоответствие нормам: наш бот дает точные ответы с ссылками на СНиП и ГОСТ. Бесплатный пилотный доступ на 30 дней, затем всего 2990 ₽/мес»

**Важное примечание** (мелким шрифтом под CTA): "*Сервис помогает находить релевантные нормативы, но для критически важных решений рекомендуется сверяться с официальными источниками"

**CTA-кнопка:** «Получить доступ к пилоту (бесплатно 30 дней)»

**Визуал:**
Скриншот Telegram-чата с примером запроса: «Какой минимальный уклон должен быть у ливневой канализации?» и ответ бота со ссылкой на СНиП 2.04.03-85, п. 2.13
Не использовать AI-генерированные изображения (снижает доверие в B2B)

### 3.2. Блок боли (2 экран)
**Заголовок:** «Каждый третий прораб сталкивается с штрафами из-за ошибок в нормативах. Вы избежите этого?»

**Подзаголовок:** «Представьте: вы на приемке объекта стоимостью 45 млн ₽. Проверяющий находит несоответствие в толщине стяжки пола. Штраф — 320 000 ₽. А решение было в 30 секундах от вас — в вашем телефоне. 78% прорабов сталкивались с подобными ситуациями в 2025 году, теряя в среднем по 2 рабочих дня на поиск нужного норматива».

**Список проблем (стиль: иконки + текст):**
- ⚠️ Приходится искать ответы на строительные вопросы в 3–4 источниках, теряя до 4 часов на объекте
- ⚠️ В обычных чатах и Telegram-ботах дают общие советы без ссылок на конкретные нормативы
- ⚠️ Нет времени на изучение толстых томов СНиП при срочных вопросах на стройке

### 3.3. Решение (3 экран)
**Заголовок:** «Задавайте вопросы в свободной форме — получайте ответы с точными ссылками на нормативы»

**Преимущества (таблица 2x2):**
| Сейчас | С «НормКонтролем» |
|------------|----------------------|
| «Спросить в WhatsApp группе прорабов» → непроверенные советы без ссылок | Написать в Telegram → получить ответ с ссылкой на конкретный пункт СНиП |
| «Полистать бумажные справочники или Excel-таблицы» → 1–2 часа поиска | 30 секунд на ответ с выдержками из актуальных нормативов |
| «Позвонить сметчику» → ожидание ответа, который может быть ошибочным | Автоматическая проверка по официальным источникам с указанием номера нормы |

**Примеры использования:**
- «Спрашиваете: Какая минимальная толщина стяжки пола по СНиП? → Получаете ответ со ссылкой на СНиП 2.03.13-88, п. 5.3»
- «Спрашиваете: В каком документе указаны требования к армированию фундамента? → Получаете перечень всех релевантных нормативов»
- «Спрашиваете: Проверь, соответствует ли проект раздела КМ требованиям? → Получаете отчет с несоответствиями и ссылками на пункты СНиП»

**Доказательство:** «Компания из Екатеринбурга избежала штрафа в 320 000 ₽ благодаря быстрой проверке требования по СНиП 3.03.01-2022, которая заняла всего 45 секунд в Telegram»

## 4. Технические требования

| Компонент | Решение | Обоснование |
|-----------|---------|-------------|
| Хостинг | Netlify (бесплатный тариф) | Бесплатный SSL, авто-CDN, мгновенный деплой |
| CMS | Статический HTML + CSS | Нет лишних зависимостей, скорость загрузки ≤1.5 сек |
| Форма | Tally.so (бесплатный тариф) | GDPR-совместимость, защита от спама, экспорт в CSV |
| Аналитика | Plausible.io ($9/мес) | Без куки, отслеживание конверсии, интеграция с Tally |
| Шрифты | System UI (без подключения внешних) | Скорость загрузки, адаптивность |
| Изображения | WebP + lazy loading | Сжатие без потерь качества, размер hero-изображения ≤150 КБ |
| Домен | normkontrol.pro ($8/год) | Короткий, запоминающийся, .pro для B2B-доверия |

**Требования к скорости:**
- Mobile Score (PageSpeed Insights) ≥ 90
- Desktop Score ≥ 95
- TTI (Time to Interactive) ≤ 1.2 сек
